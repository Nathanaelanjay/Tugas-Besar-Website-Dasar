# 🌴 TravelKu - Labuan Bajo Tour Website

Website pariwisata sederhana bertema wisata Labuan Bajo yang dibuat menggunakan HTML, CSS, JavaScript, Bootstrap, dan AOS Animation.

## 📌 Deskripsi

TravelKu merupakan website tour & travel yang menampilkan informasi wisata Labuan Bajo, layanan perjalanan, galeri, dan halaman kontak. Website ini dibuat dengan desain responsive sehingga dapat digunakan pada desktop maupun mobile.

## 🚀 Fitur

- Responsive Navbar
- Hamburger Menu Mobile
- Hero Video Background
- Animasi Scroll menggunakan AOS
- Halaman About Us
- Halaman Contact
- Google Maps Embed
- Gallery Page
- Responsive Layout
- Footer Modern

## 🛠️ Teknologi Yang Digunakan

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- AOS Animation Library
- JQuery
